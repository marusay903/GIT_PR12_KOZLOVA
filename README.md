# **Приложение "VK-Музыка"**
## *Основные возможности*
*Рекомендации и плейлисты:

+VK Микс — бесконечный плейлист, генерируемый алгоритмами на основе ваших предпочтений. Можно настраивать настроение, узнаваемость и язык треков. 
+«Сниппеты» — короткие фрагменты треков для быстрого ознакомления. 

+Тематические плейлисты по настроению, исполнителям, жанрам и другим критериям.
